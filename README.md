# Tang Nano 4K HDMI Audio Generator

## Goal
Generate a simple internal audio signal such as square wave or triangle wave inside the Tang Nano 4K FPGA and transmit it through HDMI.

## Board
- Tang Nano 4K
- Gowin FPGA
- HDMI output

## First Target
1. Confirm HDMI video output works.
2. Generate internal test audio waveform.
3. Integrate HDMI audio packet transmission.
4. Test on monitor/TV with HDMI audio support.
